# analysis/

Generated by `tools/piano-extract.mjs`. **Regenerate; never commit.**

Per source piece you get:

- `<stem>.json` — the machine-readable digest. `tools/compare.mjs` reads this directly and
  **exits 2** if a referenced bar is missing `melodySkeleton` or `harmony`.
- `<stem>-map.md` — the human-readable bar map. This is what you read when planning the
  arrangement; do not plan off the raw source file.
- `<stem>-sidecar.json` — the span sidecar you write by hand, mapping each tab span to its
  source span and declaring its mode (`quote` / `recompose` / `free`). This is an input to
  `check.mjs --map`, not a generated file, but it lives here with its digest.

Everything in this directory except this README is gitignored.
